# DOCX reference style guide

This guide describes the target Word/docx formatting for the thesis draft. Use
it when creating or editing `docs/reference.docx` for Pandoc export.

## Main Text

- Font: Times New Roman.
- Size: 14 pt.
- Alignment: justified.
- Line spacing: 1.5.
- First-line indent: 1.25 cm.
- Spacing before: 0 pt.
- Spacing after: 0 pt.

## Headings

- Font: Times New Roman.
- Weight: bold.
- Alignment: centered.
- First-line indent: none.
- Use consistent spacing before and after headings across the document.

## Heading 1

Use Heading 1 for:

- `ADNOTARE`
- `–ź–Ě–Ě–ě–Ę–ź–¶–ė–Į`
- `ANNOTATION`
- `–ě–ď–õ–ź–í–õ–ē–Ě–ė–ē`
- `–°–ü–ė–°–ě–ö –°–ě–ö–†–ź–©–ē–Ě–ė–ô`
- `–í–í–ē–Ē–ē–Ě–ė–ē`
- `–ď–õ–ź–í–ź I`
- `–ď–õ–ź–í–ź II`
- `–ď–õ–ź–í–ź III`
- `–ď–õ–ź–í–ź IV`
- `–ó–ź–ö–õ–ģ–ß–ē–Ě–ė–ē`
- `–Ď–ė–Ď–õ–ė–ě–ď–†–ź–§–ė–Į`

Recommended formatting:

- Times New Roman.
- 14-16 pt.
- Bold.
- Centered.
- Uppercase where applicable.
- No first-line indent.

## Heading 2

Use Heading 2 for numbered subsections such as `1.1`, `1.2`, `2.1`, `3.1`,
`4.1`, and similar.

Recommended formatting:

- Times New Roman.
- 14 pt.
- Bold.
- Centered.
- No first-line indent.

## Tables

- Font: Times New Roman.
- Size: 12 pt or 14 pt, depending on table width.
- First-line indent: none.
- Prefer single line spacing inside tables.
- Keep table headers bold if required by the final university template.

## Captions

Use consistent captions for figures and tables:

- Figures: `–†–ł—Ā. X.X. –Ě–į–∑–≤–į–Ĺ–ł–Ķ`
- Tables: `–Ę–į–Ī–Ľ–ł—Ü–į X.X. –Ě–į–∑–≤–į–Ĺ–ł–Ķ`

Recommended formatting:

- Times New Roman.
- 12 pt or 14 pt.
- Centered or justified according to university requirements.
- No first-line indent.

## Reference DOCX Styles

The Pandoc reference document should define at least these Word styles:

- `Normal`
- `Heading 1`
- `Heading 2`
- `Table`
- `Caption`
